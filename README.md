# wordfinder-api

To operate the solution, kindly launch it through Visual Studio or any compatible IDE that supports .NET 8.

After building, navigate the endpoints by accessing the following URL: https://localhost:5001/swagger/index.html

Within this interface, you will discover two endpoints: one for setting the matrix (POST /set-matrix) and the other for locating words within the matrix (GET /find-words).

Additionally, you can verify its functionality by executing the provided unit tests within the solution.
